# Survey Form

This project is a simple HTML survey form that collects user information and feedback. It is designed as a single-page application using only HTML.

## Features
- Collects user's name, email, and a number (age or other value)
- Dropdown selection for a custom option
- Radio buttons for yes/no questions
- Multiple checkboxes for various options (e.g., Social Media Ads, Recommendation, Location, Reputation, Price)
- Textarea for additional comments
- Form validation for required fields (name and email)

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Fill out the form and submit your responses.
